void I2CB1_Init ( uint32_t  prescale)

Initialize EUSCI_B1, I2C master.

Initialize EUSCI_B1 for I2C operation
Uses 12 MHz SMCLK clock,
7-bit address, 8 bit data
for example prescale=30 means bit rate =12MHz/30 = 400 kHz

Parameters
prescaleis clock scale, baud rate is 12MHz/prescale
Returns
none
Note
assumes 48 MHz bus and 12 MHz SMCLK
