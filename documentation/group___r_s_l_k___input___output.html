void OPT3101_ArmInterrupts ( uint32_t *  pTxChan,
uint32_t  distances[3],
uint32_t  amplitudes[3] 
)

get measurements from last measurement

Configure the OPT3101 for continuous, interrupt driven measurements. It takes about 33ms to complete one measurement. An interrupt occurs when a new measurement is complete. The ISR will set one of the three entries in the array, depending on which channel was measured. distances will have values in mm, and amplitudes will have amplitude values. It will also update the channel parameter Interrupt utilization
1) Start using OPT3101_StartMeasurementChannel(channel);
2) ISR updates values in the arrays

Parameters
pTxChanto a variable for the channel value
distancesarray for each of the three channels
amplitudesarray for each of the three channels
Returns
none

◆ Reflectance_Position()

int32_t Reflectance_Position ( uint8_t  data)

Perform sensor integration.

Calculate the weighted average for each bit:
Position varies from -332 (left) to +332 (right), with units of 0.1mm.

8 element arrays
bit7bit6bit5bit4bit3bit2bit1bit0
Weight-332-237-142-4747142237332
Mask 0x010x020x040x080x100x200x400x80

count = 0
sum = 0
for i from 0 to 7
if (data&Mask[i]) then count++ and sum = sum+Weight[i]
calculate position = sum/count

Parameters
datais 8-bit result from line sensor
Returns
position in 0.1mm relative to center of line
Note
returns 333 if data is zero (off the line)

Initialize a GPIO pin for output, which will be used to trigger the ultrasonic sensor. Initialize the input capture interface, which will be used to take the measurement.
